# Branding Contract

## Required Public Identity

- Short name: `GBBox`
- Long name: `GameBase Box`
- Positioning: local-first desktop library for GameBase-style databases.
- Supported imports in 0.1.0: Commodore 64, Atari 800, Atari 2600.
- Forward-looking copy: more GameBase platforms coming soon.

## Replacement Rules

- Replace visible `64Box` product names with `GBBox`.
- Replace expanded product descriptions with `GameBase Box`.
- Replace logos or assets that visibly say `64Box`.
- Keep `Commodore 64`, `C64`, and `GameBase64` where they describe the supported C64 platform or source database.
- Keep `64Box` only when describing the old repository/project history or redirect.

## Verification

- Run a case-sensitive and case-insensitive branding search over tracked files.
- Review splash/about/settings/title/README/release-visible copy.
- Verify icons/logos no longer display 64Box.
